# 06. 빌드 및 실행 가이드

이 문서는 CrimsonRunner를 빌드하고 실행하는 방법을 설명합니다.

## 요구 사항

- Windows
- Visual Studio 2022
- MSVC v143 C++ 빌드 도구
- Windows SDK
- DirectX 11 지원 환경

## Visual Studio에서 열기

다음 파일을 엽니다.

`C:\Users\yejun\source\repos\CrimsonRunner\CrimsonRunner.slnx`

또는 프로젝트 파일을 직접 열 수도 있습니다.

`C:\Users\yejun\source\repos\CrimsonRunner\CrimsonRunner.vcxproj`

## 빌드 설정

권장 설정은 다음입니다.

- Configuration: `Debug`
- Platform: `x64`

## 명령줄 빌드

PowerShell에서 다음 명령을 실행합니다.

```powershell
& 'C:\Program Files\Microsoft Visual Studio\2022\Professional\MSBuild\Current\Bin\MSBuild.exe' `
  'C:\Users\yejun\source\repos\CrimsonRunner\CrimsonRunner.vcxproj' `
  /p:Configuration=Debug `
  /p:Platform=x64 `
  /m
```

## 빌드 결과

빌드에 성공하면 실행 파일이 생성됩니다.

`C:\Users\yejun\source\repos\CrimsonRunner\x64\Debug\CrimsonRunner.exe`

## 실행 전 확인 사항

실행 파일의 작업 디렉터리는 프로젝트 루트여야 합니다.

왜냐하면 텍스처와 셰이더를 다음처럼 상대 경로로 읽기 때문입니다.

- `Textures/...`
- `Shaders/BasicColor.hlsl`

Visual Studio 프로젝트에는 작업 디렉터리가 `$(ProjectDir)`로 설정되어 있습니다.

## 흔한 문제

### 화면은 뜨는데 이미지가 안 보임

작업 디렉터리가 프로젝트 루트가 아닐 가능성이 큽니다. Visual Studio에서 실행하거나, 실행 파일을 프로젝트 루트 기준으로 실행하세요.

### 빌드가 안 됨

Visual Studio C++ 빌드 도구와 Windows SDK가 설치되어 있는지 확인하세요.

### 픽셀아트가 흐릿함

현재 코드는 Point 필터를 사용하므로 정상 빌드라면 선명해야 합니다. 만약 흐릿하다면 다른 실행 파일을 실행 중인지 확인하세요.
