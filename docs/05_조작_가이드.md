# 05. 조작 가이드

이 문서는 사용자가 어떤 키를 누르면 게임에서 어떤 일이 일어나는지 설명합니다.

## 기본 조작표

| 키 | 동작 | 설명 |
|---|---|---|
| `Space` | 점프 | 플레이어가 위로 뛰어오릅니다. 지상 장애물을 피할 때 사용합니다. |
| `↑` | 점프 | `Space`와 같은 점프입니다. 방향키 조작을 선호하는 사용자를 위한 입력입니다. |
| `↓` | 슬라이드 | 누르고 있는 동안 플레이어가 낮게 엎드립니다. 위쪽 벽 장애물을 피할 때 사용합니다. |
| `R` | 재시작 | 게임오버 상태에서만 작동합니다. 플레이 중에는 작동하지 않습니다. |
| `P` | 점수 증가 | 디버그용입니다. 점수를 1000 올려 배경 전환을 빠르게 확인할 수 있습니다. |

## 점프 조작

점프는 `Space` 또는 `↑` 키를 누르면 실행됩니다.

```text
Space 또는 ↑ 입력
→ 플레이어가 바닥에 있는지 확인
→ 바닥에 있으면 위쪽 속도 부여
→ 중력에 의해 다시 내려옴
→ 바닥에 닿으면 달리기 상태로 복귀
```

## 짧은 점프

점프 키를 짧게 누르고 떼면 점프 높이가 줄어듭니다. 이 기능은 점프를 더 세밀하게 조작하게 해줍니다.

## 슬라이드 조작

`↓` 키를 누르고 있으면 슬라이드합니다.

```text
↓ 누름
→ 플레이어 높이가 낮아짐
→ 위쪽 벽 아래를 지나갈 수 있음
↓ 뗌
→ 다시 달리기 상태로 복귀
```

## 재시작 조작

`R` 키는 게임오버 상태에서만 작동합니다.

이렇게 제한한 이유는 플레이 중 실수로 `R`을 눌러 게임이 갑자기 초기화되는 버그를 막기 위해서입니다.

## 장애물별 피하는 방법

| 장애물 | 피하는 방법 | 설명 |
|---|---|---|
| 경찰 | 점프 | 경찰 몸통 판정을 뛰어넘습니다. |
| 개 | 점프 | 낮게 있는 개를 점프로 피합니다. |
| 벽 | 점프 | 지상 벽을 넘어갑니다. |
| 위쪽 벽 | 슬라이드 | 플레이어 머리 위를 지나가는 벽 아래로 미끄러집니다. |

## 게임오버 시 화면

충돌하면 다음 일이 일어납니다.

1. 게임 진행이 멈춥니다.
2. 화면이 어두워집니다.
3. 감옥이 플레이어 위로 떨어집니다.
4. `GAME OVER` 문구가 표시됩니다.
5. `R RESTART` 안내가 표시됩니다.
6. `R`을 누르면 처음부터 다시 시작합니다.
